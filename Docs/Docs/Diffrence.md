## Difference Between Value Type and Reference Type

| Value Type | Reference Type |
|-------------|---------------|
| Stored on **stack memory** | Stored on **heap memory** |
| Contains the **actual value** | Contains a **reference (pointer) to the value** |
| Cannot contain **null values** (unless using nullable types) | Can contain **null values** |
| Memory is allocated at **compile time** | Memory is allocated at **run time** |

---

## Difference Between Stack and Heap

| Stack | Heap |
|------|------|
| Values are stored **one on another (LIFO structure)** | Values are stored in **random order in a large memory space** |
| Used for **value types** | Used for **reference types** |
